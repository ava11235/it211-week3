Learning Outcomes

Upon successful completion of this chapter, students will be familiar with and able to apply the following concepts in their programs

  1. Tables
  2. Images
  3. Links
  4. Special Characters

Notes

Tables are used to organize data in rows and columns.

Imagesimprove the visual appeal of a web page.

HTML links (Hyperlinks) allow the user to get from one page to another by clicking them.

Special characters are a way of writing entities, such as the © symbol.
